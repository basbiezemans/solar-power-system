# Solar Power System
App that helps you size a solar power system.

## Usage

### Build
```bash
npm run build         # Compile TypeScript to dist/
npm run build:watch   # Watch mode for development
```

### Testing
```bash
npm run test:run       # Run unit tests once
npm run test           # Watch mode for unit tests
npm run test:watch     # Continuous testing during development
npm run e2e            # Run end-to-end tests
npm run e2e:ui         # Run E2E tests with UI
npm run e2e:debug      # Run E2E tests in debug mode
```

### Running the Application
```bash
npm run preview
```

## Browser Compatibility
- ES2020 target
- Browser must support ES modules (ESM)
- Browser must support WebSocket connections
